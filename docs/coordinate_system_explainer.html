像素 + 深度如何变成三维点

这个页面用你的 Orbbec RGB-D 流程解释相机坐标系：mask 先决定哪些像素属于目标，depth 给出每个像素的距离，再用相机内参把目标像素反投影成三维点云。

当前三维点

这里使用你相机当前读到的近似内参：fx=610.85、fy=610.65、cx=644.34、cy=403.04。

X 右正左负
Y 下正上负
Z 前方深度

反投影公式

X = (u - cx) * Z / fx
Y = (v - cy) * Z / fy
Z = depth

每一个 mask 内的有效深度像素都会套用这三个公式，所以会得到很多个三维点，也就是目标点云。左侧相机坐标系里的三维点，对应右侧图像平面里的像素点。

颜色说明

绿色区域：mask 目标像素
红色轴：X，向右为正
绿色轴：Y，向下为正
蓝色轴：Z，向前为正

程序里的步骤

  1. HSV 或模型生成 mask。
  2. 只保留 mask > 0 且深度有效的像素。
  3. 读取这些像素的 u, v, depth。
  4. 用内参反投影成 X, Y, Z。
  5. 对点云做中心、尺寸、体积和抓取点计算。
